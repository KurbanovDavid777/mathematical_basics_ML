## Описание проекта

В этом репозитории собраны лабораторные работы по математическим методам и анализу данных для курсов по машинному обучению. Цель проекта — закрепление теоретических знаний через практическую реализацию алгоритмов и методов анализа данных.

### Содержание папки `task`

- **Lab 1:** Алгоритмы разложения матриц (PCA и Kernel PCA)
- **Lab 2:** Численные методы оптимизации (BFGS, L-BFGS, численные градиенты)
- **Lab 3:** Одномерный анализ данных для улучшения качества данных
- **Lab 4:** Многомерный анализ данных для выявления зависимостей и улучшения качества данных

### Используемые технологии

- Python 3.x
- Jupyter Notebook
- numpy, pandas, matplotlib, seaborn
- scikit-learn

---

## Project Description

This repository contains laboratory works on mathematical methods and data analysis for machine learning courses. The goal is to reinforce theoretical knowledge by practical implementation of algorithms and data analysis techniques.

### Contents of the `task` folder

- **Lab 1:** Matrix decomposition algorithms (PCA and Kernel PCA)
- **Lab 2:** Numerical optimization methods (BFGS, L-BFGS, numerical gradients)
- **Lab 3:** Univariate data analysis for data quality improvement
- **Lab 4:** Multivariate data analysis for detecting dependencies and improving data quality

### Technologies used

- Python 3.x
- Jupyter Notebook
- numpy, pandas, matplotlib, seaborn
- scikit-learn

## Инструкция по запуску

1. Клонировать репозиторий: / Clone the repository:

```bash

```

2. Перейти в папку проекта: / Navigate to the project folder:

```bash
cd mathematical_basics_ML
```

3. Открыть Jupyter Notebook: / Launch Jupyter Notebook:

```bash
jupyter notebook
```

4. Перейти в папку task и открыть нужную лабораторную работу.
